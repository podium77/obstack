# Conception du système d'autorisation

Conçois le système RBAC complet.

Produis :

- les rôles ;
- les permissions ;
- leur portée ;
- les règles d'héritage éventuelles.

Distingue clairement :

- les permissions globales ;
- les permissions liées à une entreprise ;
- les permissions liées à un environnement ;
- les permissions liées à la propriété d'une ressource.

Précise également :

- les règles de contrôle côté API ;
- les vérifications obligatoires côté backend ;
- les protections contre l'escalade de privilèges ;
- les cas de tentative d'accès non autorisé.
