# Implémentation backend

À partir des documents validés :

Propose l'implémentation backend complète.

Inclure :

- les modèles ;
- les services métier ;
- les repositories ;
- les contrôleurs ;
- les DTO ;
- les validations ;
- les middlewares ou policies d'autorisation ;
- les endpoints API.

Pour chaque endpoint préciser :

- qui peut appeler l'API ;
- les vérifications nécessaires ;
- les erreurs à retourner.

Respecter une architecture propre (Clean Architecture ou équivalent).
