# Étude d'architecture

À partir des règles métier fournies, réalise une étude d'architecture.

Produis :

1. Le modèle de domaine.
2. Les entités nécessaires.
3. Les relations entre les entités.
4. Les contraintes d'intégrité.
5. Les règles de sécurité.
6. Les contraintes de multi-tenancy.
7. Les cas limites à gérer.

Ne génère aucun code pour le moment.

Fournis :
- un diagramme de classes textuel ;
- les cardinalités ;
- les contraintes de suppression ;
- les règles d'héritage ou de composition si nécessaires.
