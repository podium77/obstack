# Implémentation frontend

Propose l'impact sur le frontend.

Décrire :

- les nouveaux écrans ;
- les formulaires ;
- les listes ;
- les composants ;
- les menus visibles selon les droits.

Important :

Le masquage des boutons ou menus n'est qu'une amélioration ergonomique.

La sécurité doit toujours être vérifiée par le backend.
